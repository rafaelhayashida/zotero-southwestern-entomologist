# zotero-southwestern-entomologist
 Custom CSL style adapted for the journal Southwestern Entomologist.

## Installation

1. Download the `.csl` file
2. Open Zotero
3. Go to Preferences → Cite → Styles
4. Click "+" and import the file

## Notes
Based on the reference style required by the Southwestern Entomologist Instructions to Authors. The style follows a
Chicago-style reference list (year at the end) with author–date in-text citations, matching the journal's Google Scholar /  Chicago requirement.

##Feedback

(the item type and its fields) so it can be fixed.
